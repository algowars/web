Write a function that returns the numbers from `1` to `n` as a string separated by commas.

Replace certain values if any of these conditions are met:

- If the value is a multiple of `3`: use the value "Fizz"
- If the value is a multiple of `5`: use the value "Buzz"
- If the value is a multiple of `3` and `5`: use the value "FizzBuzz"

**Example 1**

**Input**: `15`  
**Output**: `"1,2,Fizz,4,Buzz,Fizz,7,8,Fizz,Buzz,11,Fizz,13,14,FizzBuzz"`
